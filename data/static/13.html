Processing Libraries

Libraries make it possible for sketches to do things beyond the core Processing code.

They can enable new things like playing sounds, doing computer vision, and working with advanced 3D geometry. Contributed libraries can be installed via Sketch > Import Library > Add Libraries...

 

Processing Tools

Tools extend the PDE to help make creating sketches easier by providing interfaces for tasks like selecting colors. Contributed tools can be installed via Tools > Add Tool...
